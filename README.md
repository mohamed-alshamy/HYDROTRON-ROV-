## 📌 About The Project

**HYDROTRON ROV** is an underwater remotely operated vehicle (ROV) designed for underwater exploration, inspection, and manipulation tasks.

The project focuses on developing an intelligent underwater robotic system that combines embedded systems, robotics, computer vision, and AI technologies to perform real-time underwater operations.

The ROV is capable of underwater navigation, monitoring, object detection, object tracking, and performing manipulation tasks using robotic grippers.

---

## 🚀 Features

- 🌊 Underwater remotely controlled operation
- 🎮 Real-time vehicle control
- 📷 Live underwater camera monitoring
- 🧠 AI-based object detection and tracking
- 🔍 Real-time image processing
- ⚙️ Embedded control system
- 🔌 Motor and actuator control
- 🤖 Robotic gripper manipulation
- 📡 Communication between surface station and ROV

# 🛠️ Hardware Components

## 🔹 Propulsion System

- **6 × T200 Thrusters**

Used for underwater movement and navigation:

- Forward / Backward movement
- Vertical movement
- Rotation control
- Vehicle stabilization


- **6 × Electronic Speed Controllers (ESCs)**

Responsible for controlling thruster speed and direction.

## 🔹 Robotic Gripper System

The ROV is equipped with **2 pneumatic robotic grippers** for underwater manipulation tasks.

The gripper system consists of:

- **2 × Robotic Grippers**
  - Each gripper is controlled by a dedicated pneumatic valve.
  - Each valve controls the opening and closing movement of its corresponding gripper.

- **1 × Rotation Motor**
  - Responsible for controlling the grippers' rotation.
  - Provides vertical and horizontal rotation movement.

- **1 × Pneumatic Valve**
  - Controls the rotation mechanism through the motor system.

- **3 × Pneumatic Valves Total**
  - 2 valves for gripper operation.
  - 1 valve for gripper rotation control.

All pneumatic valves are connected to an air compressor to provide the required pressure for the gripper operation.

The grippers support:

- Opening and closing
- Vertical rotation
- Horizontal rotation
