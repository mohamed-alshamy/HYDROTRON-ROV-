## 📌 About The Project

**HYDROTRON ROV** is an underwater remotely operated vehicle (ROV) designed for underwater exploration and control applications.

The project focuses on building an intelligent underwater robotic system capable of performing tasks such as underwater navigation, monitoring, and data collection using embedded systems, sensors, and robotics technologies.

---

## 🚀 Features

- 🌊 Underwater remotely controlled operation
- 🎮 Real-time control system
- 📷 Camera-based underwater monitoring
- ⚙️ Embedded control system
- 🔌 Motor and actuator control
- 📡 Communication between surface station and ROV
- 🧠 Robotics and automation concepts

---

## 🛠️ Hardware Components

- Microcontroller / Embedded Controller

- Motor Drivers
- Underwater Camera
